# Portfólio Pessoal Profissional | 2026

Link: https://johnnycvz.github.io/Portif-lio---Projeto-Integrado-III---Professor-Fernando-Datorre/

Este projeto consiste no desenvolvimento de um site portfólio moderno e imersivo, focado na intersecção entre a **Engenharia de Computação** e a **Produção Audiovisual**. O design foi concebido com uma estética de alto nível, utilizando paleta de cores *Midnight Blue* e *Gold*, inspirado em referências globais do *Awwwards* e *Lusion*.

## Tecnologias Utilizadas
O projeto foi construído puramente com tecnologias Web nativas, focando em performance e sem o uso de frameworks prontos:
- **HTML5**: Estruturação semântica e acessível.
- **CSS3**: Estilização avançada, variáveis (Root), Glassmorphism, Flexbox/Grid e animações complexas.
- **JavaScript (Vanilla)**: Implementação de efeitos de Parallax suave, Cursor customizado interativo e API de Intersection Observer para revelação de seções ao rolar (Scroll Reveal).

---

## Destaques do Projeto
- **Manifesto Pessoal**: Uma introdução narrativa que une o rigor da UNIFEV à fluidez do audiovisual.
- **Galeria Autoral**: Exposição de fotografias em formato 9:16 (Nikon D40X e Sony W620).
- **Efeito Parallax**: Fundo dinâmico com profundidade sutil.
- **Responsividade**: Adaptado para diferentes resoluções de tela.
